## 专题一：人工智能与新闻传播

### 一、AIGC（AI Generated Content）

#### 1. 核心概念
- **定义**：利用AI技术自动生成文本、图片、音频、视频等内容。
- **关键词**：AIGC、生成式AI、自动化内容生产

#### 2. 新闻业应用（★★★）
| 应用环节 | 具体功能 |
| :--- | :--- |
| **写作** | 机器人写作（财经、体育、天气等） |
| **采访** | 智能语音转写、自动摘要 |
| **编辑** | 智能校对、自动排版 |
| **分发** | 算法推荐、个性化推送 |
| **审核** | 内容审核、虚假信息识别 |

#### 3. 对新闻业的影响（★★★★★）
- **积极影响**：
    - **提高效率**：**大幅提升新闻生产效率**
    - **降低成本**：降低人力与时间成本
    - **个性化**：实现内容的精准推送
    - **扩展范围**：覆盖人力难以企及的报道领域
- **消极影响**：
    - **就业冲击**：威胁新闻从业者岗位
    - **虚假信息**：可能被用于制造假新闻、深度伪造
    - **版权伦理**：版权归属、责任界定不清
    - **专业主义削弱**：**削弱新闻专业主义与人文关怀**

#### 4. 伦理挑战（★★★★）
- **真实性**：AI生成内容是否可信？
- **版权**：生成内容的版权归谁？（平台？用户？AI开发者？）
- **责任**：AI报道出错，谁来负责？（算法？开发者？使用者？）
- **偏见**：AI算法是否存在固有偏见，加剧社会不公？

### 二、ChatGPT / 大语言模型

#### 1. 核心特点
- **强大理解力**：自然语言理解与生成能力
- **流畅生成**：文本生成流畅，逻辑性强
- **多任务处理**：可完成问答、翻译、摘要、创作等多种任务

#### 2. 对新闻传播的影响（★★★★）
- **信息获取**：**改变用户获取信息的方式**（从搜索到对话）
- **搜索引擎**：**冲击传统搜索引擎的霸主地位**
- **内容生产**：改变内容创作模式，人机协同成为可能
- **信息真实性**：引发对**信息真实性**和**深度伪造**的更大担忧

## 专题二：短视频传播

### 一、发展现状
- **代表平台**：抖音、快手（国内）；TikTok（海外）；B站（中长视频）
- **关键数据**：用户超**10亿**，日均使用时长超**2小时**，最大流量入口

### 二、传播特征（★★★★）

#### 1. 内容特征
- **短小精悍**：时长15秒-3分钟
- **碎片化**：内容零散、不完整
- **视觉化**：以视觉冲击力为核心
- **娱乐化**：娱乐内容占主导

#### 2. 传播特征
- **算法推荐**：**基于算法的精准分发**是核心机制
- **社交分享**：依托社交网络进行裂变式传播
- **圈层化**：不同群体形成各自的内容偏好圈
- **病毒式传播**：优质内容易实现快速、大规模扩散

#### 3. 用户特征
- **年轻化**：以Z世代为主力
- **下沉化**：向三四线城市及农村地区渗透
- **参与化**：**用户既是消费者也是生产者**（UGC）

### 三、社会影响（★★★★★）

| 积极影响 | 消极影响 |
| :--- | :--- |
| 丰富信息传播形式 | **信息茧房**：算法推荐加剧信息窄化 |
| 降低内容创作门槛 | **注意力碎片化**：深度思考能力下降 |
| 推动文化传播与交流 | **低俗内容泛滥**：内容质量参差不齐 |
| 带动直播电商等经济发展 | **网络沉迷**：过度依赖，影响身心健康 |

### 四、与传播学理论的关联（★★★★★）
- **议程设置**：短视频平台（热搜、推荐）成为**新的议程设置主体**
- **使用与满足**：用户主动选择短视频，满足**娱乐、社交、信息获取**等需求
- **培养理论**：长期观看算法推荐内容，会**潜移默化地影响用户世界观**，且效果被算法放大

## 专题三：算法推荐与信息茧房

### 一、算法推荐原理

#### 1. 推荐算法类型
- **协同过滤**：基于“与你相似的人喜欢什么”进行推荐
- **内容推荐**：基于“你过去喜欢的内容特征”进行推荐
- **混合推荐**：综合多种算法，效果最优

#### 2. 推荐逻辑
1.  **分析行为**：记录点击、浏览、停留、点赞、评论等
2.  **构建画像**：形成用户兴趣、人口属性等标签
3.  **匹配内容**：将用户画像与内容标签进行匹配
4.  **个性化推荐**：推送最可能吸引用户的内容

### 二、信息茧房（★★★★★）
- **提出者**：**凯斯·桑斯坦**
- **核心概念**：人们只关注自己感兴趣或认同的信息，如同被困在蚕茧中，导致视野狭窄、认知固化。
- **形成原因**：
    - **算法推荐**：**“投其所好”**，强化既有偏好
    - **选择性接触**：用户主动回避异质信息
    - **社交圈层化**：社交网络同质化，信息源单一
- **负面影响**：
    - **认知偏见**：强化刻板印象，形成偏激观点
    - **社会极化**：不同群体间共识减少，对立加剧
    - **公共讨论削弱**：缺乏多元信息，理性对话空间被压缩

### 三、算法治理（★★★★）
- **治理措施**：
    - **算法透明**：公开推荐逻辑，接受监督
    - **用户选择权**：提供“不感兴趣”、“关闭推荐”等选项
    - **价值引导**：将主流价值观融入算法设计
    - **法规监管**：完善相关法律法规
- **相关法规**：**《互联网信息服务算法推荐管理规定》**（要求提供关闭算法推荐选项）

## 专题四：媒体深度融合

### 一、背景
- **技术驱动**：互联网、移动互联网普及，新媒体冲击
- **政策推动**：国家战略，要求“推动媒体深度融合”

### 二、融合路径（★★★★）

#### 1. 中央厨房模式
- **概念**：**“一次采集、多种生成、多渠道传播”**
- **特点**：统一指挥、统一采集、多元分发
- **代表**：**人民日报中央厨房**

#### 2. 县级融媒体中心
- **目标**：打通基层传播 **“最后一公里”**
- **功能**：**“新闻+政务+服务”**
- **特点**：贴近基层，服务群众

#### 3. 全媒体传播体系
- **概念**：构建 **“全程、全息、全员、全效”** 的传播格局

### 三、挑战
- **体制机制**：传统媒体体制僵化，难以适应新媒体
- **人才短缺**：缺乏懂技术、懂运营的复合型人才
- **技术投入**：技术更新快，资金投入大
- **商业模式**：盈利模式不清晰，自我造血能力不足

## 专题五：国际传播能力建设

### 一、现状与挑战
- **不平等格局**：**“西强东弱”**，西方媒体主导国际话语权
- **中国挑战**：面临 **“中国威胁论”** 等负面舆论，文化差异导致误解

### 二、讲好中国故事（★★★★★）
- **策略**：
    - **内容创新**：寻找具有**人类共通情感**的故事
    - **渠道拓展**：利用TikTok、YouTube等**新媒体平台**出海
    - **话语创新**：构建**融通中外**的中国话语体系
    - **人才培养**：培养具有国际视野的传播人才
- **实践案例**：**李子柒**（文化输出）、**TikTok**全球化、**CGTN**（中国国际电视台）

### 三、提升国际话语权路径
- **增强传播能力**：提升内容质量与传播技巧
- **创新话语方式**：用国际受众听得懂、易接受的方式讲述
- **拓展传播渠道**：借力海外平台，建设自有平台
- **加强人文交流**：通过民间交流，增进理解与互信

## 专题六：网络舆论与舆情治理

### 一、网络舆论特点（★★★★）
- **即时性**：形成快、传播快、变化快
- **多元性**：主体、观点、渠道多元
- **情绪化**：**情绪传播快于理性**，易被煽动和放大
- **不确定性**：走向难预测，易反转，难控制

### 二、
