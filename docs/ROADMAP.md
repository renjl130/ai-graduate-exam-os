# 产品与工程路线图（ROADMAP）

> 最近更新：2026-07-16

## P0 — 商业化前稳定与治理

### P0.1 长期架构主线 ✅

- Cloudflare 确定为 Tier A 商业生产主线；
- FastAPI 确定为 Tier B 自托管兼容；
- 新功能不再默认双写；
- FastAPI 不作为实时热备。

### P0.2 API 契约与质量门禁

- 建立 88 个共享路由契约；
- 统一认证、错误、分页和 request_id；
- 建立单元、API 集成、契约和关键 E2E；
- CI 强制 typecheck、test、build。

### P0.3 安全

- 登录、注册、AI、上传和社区限流；
- CORS allowlist；
- Token、刷新和会话撤销方案；
- PDF query token 整改；
- 用户数据导出和注销闭环。

### P0.4 可观测性与数据安全

- 结构化日志和错误追踪；
- API、AI、D1/KV 指标和告警；
- 备份、恢复和演练；
- AI 成本与配额监控。

### P0.5 工程债

- 测试后拆分 Worker route 和 FastAPI `main.py`；
- 收敛前端直接 fetch；
- 收敛双 CSS Token；
- 清理确认无用代码和依赖；
- 统一版本号和文档状态。

- [x] 建立顶部信息渐进式响应规则；

## P1 — 核心学习闭环

### P1.1 学习计划产品化

- 统一任务实体和服务端数据源；
- 周/月视图、重复任务、优先级和拖拽排程；
- 计划与专注、复习和模考联动。

### P1.2 统一复习队列

- 合并知识、闪卡、词汇和错题复习入口；
- 版本化间隔算法；
- 建立复习债务和今日收件箱。

### P1.3 模考与错题闭环

- 草稿、断点恢复和真实计时；
- 自动/人工评分；
- 考试结果自动进入错题和薄弱点；
- 预测展示计算依据和置信度。

### P1.4 AI 可靠性

- 会话历史 UI、引用、超时和取消；
- RAG 来源引用；
- 成本配额和降级；
- AI 输出质量评测集。

### P1.5 用户生命周期

- 邮箱验证、找回密码、注销和数据导出；
- 多设备会话；
- 用户设置服务端同步；
- 隐私和条款同意记录。

### P1.6 UI 与正式路由

- 统一 Button/Input/Card/Modal/Empty/Error/Skeleton；
- 可访问性和视觉回归；
- hash 页面渐进迁移正式路由；
- 建立可分享深链接。

## P2 — 内容与平台

- 真正向量检索、混合检索和 RAG；
- 内容管理后台、审核、发布和回滚；
- 社区评论、举报、审核和反滥用；
- 通知提醒和运营配置；
- PWA、弱网和离线只读。

## P3 — 商业化与生态

- 多院校、多专业和多考试体系；
- 订阅、权益、配额和支付；
- 移动客户端；
- 教师、机构、班级和课程；
- 多 Agent 学习教练。

## 当前建议顺序

1. 共享 API 契约；
2. 自动化测试和 CI 门禁；
3. 安全、监控和备份；
4. 计划/复习/模考闭环；
5. 真实 RAG 和内容后台；
6. 商业化扩展。
